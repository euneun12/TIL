## Git

1. 저장소(repository) 만들기 -  git이 관리하는 폴더(directory)

   이미 init한 repo에서 또 init하지 않는다!

   `$ git init` 

2.  파일을 스테이징 하기

   `$ git add 파일명`

3.  파일을 커밋하기

   `$ git commit -m '커밋 메세지 내용(변경사항을 요약해서 적음)`

- 상태확인하기

  `$ git status`

- 내가 누군지 정보 입력하기

  `$ git config --global user.name 유저이름`

  `$ git config --global user.email 유저이메일`
